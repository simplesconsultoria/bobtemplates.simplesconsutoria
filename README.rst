Uso
===

Após a instalação do mr.bob: ::

    $ pip install mr.bob==0.1a9

Podemos executar o comando: ::

    $ mrbob -O nome_do_addon --config https://raw.github.com/simplesconsultoria/bobtemplates.simplesconsultoria/master/test_answers_addon.ini https://github.com/simplesconsultoria/bobtemplates.simplesconsultoria/archive/master.zip#bobtemplates.simplesconsultoria-master/bobtemplates/simplesconsultoria/addon

Para gerar o novo addon a partir do master do pacote.
